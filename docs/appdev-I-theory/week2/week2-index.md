# Week 2

<SessionResources :week="2" />

This week builds the **foundations of how content is represented and displayed on the Web**.

**CSS** controls visual styling with special focus on **selectors and specificity**, as these concepts are generally asked in PYQs.

Some concepts of **HTML** which are necessary for basic page structuring.

By the end of this week, you should be able to:
- Convert numbers between different number systems, used internally by machines
- Understand how text is encoded and stored in computers
- Write basic HTML to structure content
- *Which CSS selector styles actually applied* by browser when multiple rules exist

### Topics Covered

- [Number Systems](2-number-systems.md)  
  Decimal, binary, octal, hexadecimal conversions

- [Character Encoding](2-ascii-unicode.md)  
  ASCII, Unicode, UTF-8, and how text is represented in memory

- [HTML Basics](2-html-basics.md)  
  Structure of an HTML document, Block vs inline elements and semantic structure

- [HTML Elements](2-html-elements.md)  
  Basic tags for structuring pages like lists, links & images.

- [HTML Tables](2-html-tables.md)  
  Tabular data representation using HTML

- [CSS Important Topics: Selectors & Specificity Order](2-css-selector-specificity.md)  
  How CSS decides which style rules win, type of selectors used

- [CSS Extra Selectors](2-css-extra.md)  
  Advanced selectors for list, tables and pseudo-classes
